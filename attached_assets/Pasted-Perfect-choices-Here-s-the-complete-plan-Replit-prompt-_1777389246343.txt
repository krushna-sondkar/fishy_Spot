> ### ⚙️ BACKEND — Node.js + Express
>
> **`POST /api/orders`**
> - Accept full order object (customer details, items, address, slot, total)
> - Generate order ID: `FS-` + Date.now()
> - Save to `orders.json` file
> - Return: `{ success: true, orderId: "FS-..." }`
>
> **`GET /api/orders`**
> - Return all orders from orders.json (basic admin view)

> - Return the fish catalogue as JSON
